# README - OptiCam

## 📷 Sobre o Projeto

O **OptiCam** é uma plataforma fictícia de fotografia inteligente desenvolvida para apresentar um sistema moderno de cadastro e login com uma interface visual elegante e interativa.

O projeto utiliza:

- HTML5
- CSS3
- JavaScript

A aplicação possui:

- Página inicial interativa
- Sistema de login
- Sistema de cadastro
- Slideshow de imagens
- Validação de formulários
- Interface moderna utilizando Flexbox

# 🎨 Funcionalidades

## 🏠 Página Inicial

A página principal apresenta:

- Nome da plataforma
- Descrição do sistema
- Botões de acesso:
  - Cadastro
  - Login
- Slideshow interativo de imagens

---

## 🔐 Sistema de Login

O login possui:

- Campo de e-mail
- Campo de senha
- Opção "Lembrar-me"
- Link "Esqueci a senha"
- Validação de formulário via JavaScript

### Validações

- Verifica se o e-mail foi preenchido
- Verifica se o e-mail possui:
  - "@"
  - ".com"
- Verifica se a senha possui pelo menos 8 caracteres

---

## 📝 Sistema de Cadastro

A página de cadastro possui:

- Campo de e-mail
- Campo de senha
- Botão de cadastro
- Botão de cancelamento

### Validações

- Campos obrigatórios
- E-mail válido
- Senha mínima de 8 caracteres

Após o cadastro:

```javascript
alert(`Cadastro efetuado com sucesso! Bem-vindo ao OptiCam.`);
```
